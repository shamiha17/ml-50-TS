
## 📊 Climate Time Series Analysis

This project analyzes daily climate data using Python. It explores trends in temperature, humidity, wind speed, and pressure over time.

### 🔹 Features

* Data preprocessing (date formatting, indexing)
* Data visualization using plots
* Basic time series techniques (rolling mean, lag features)

### 🛠️ Tools

* Python
* Pandas
* Matplotlib

### 🎯 Goal

To understand climate patterns and prepare data for time series modeling.
